Uma simples aplicação de página web

Aqui você verá o conceito SPA sendo aplicado na prática, pois nelas não há a necessidade de se fazer requisições para carregamento de novas páginas. Onde na primeira requisição, todo o HTML, CSS e JavaScript necessários seram carregados de uma vez. Apartir deste momento os componentes seram gerenciados e carregados através de rotinas JavaScript retirando a necessidade de requisições para o servidor com a finalidade de obter o novo conteúdo a ser renderizado. A vantagem das aplicações SPA é a otimização e performance das páginas web do lado do cliente e permitir tráfego de dados mais leve entre cliente e servidor.
